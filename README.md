This is a lightweight, responsive personal portfolio built with HTML/CSS/JS. It includes About, Experience, Skills, Education, Achievements, and a contact form stub.

## Quick start
1. Open `index.html` in a browser.
2. Replace links and content in `index.html` as needed.
3. To deploy:
   - **GitHub Pages**: push this folder to a repo and enable Pages (main branch, root).
   - **Netlify/Vercel**: drag-and-drop the folder in the dashboard.

## Customization
- Colors and typography: edit `styles.css` variables.
- Contact form: integrate Formspree, Power Automate, or backend of choice.
- Add projects: create a new `<section id="projects">` and cards.

## Assets
- Add a profile photo to `assets/portrait.jpg` and reference it in the header if desired.
